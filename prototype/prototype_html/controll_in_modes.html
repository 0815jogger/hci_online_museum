Modes

All the shown controlls have a different purpose in each mode. The modes are "Exploration Mode" and "Education Mode".

NEXT

Education Mode

In this Mode the Object with information layers is shown.

You can explore informative bordered areas the object to see informations about that. The controll cross supporting now the possibility to switch between the existing information layers. The push event will still zooming in, but in a different way. You can now zoom information layers to get more detail informations about the layers above.

 

Exploration Mode

In this Mode the bare Object without any layers or functionality is shown.

You can explore the object by rotating and zooming it.
